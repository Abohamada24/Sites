## ✨ المميزات

### 🔍 البحث الذكي
- دعم متعدد المصادر (OpenSubtitles, Subscene, YouTube)
- بحث بالاسم، IMDB ID، أو رابط الفيديو
- تصفية حسب اللغة والجودة والتقييم

### ⬇️ تحميل قوي
- تحميل سريع مع شريط تقدم
- استئناف التحميل عند الانقطاع
- تحميل دفعات متعددة
- دعم الملفات المضغوطة (ZIP, RAR)

### 🛠️ أدوات متقدمة
- تحويل بين الصيغ (SRT, VTT, ASS, SSA)
- تنظيف تلقائي (إزالة الإعلانات والوسوم)
- دمج ملفات متعددة
- استخراج معلومات الترجمة

### 🌐 دعم اللغات
- العربية
- الإنجليزية
- الفرنسية
- الإسبانية
- وأكثر من 50 لغة
